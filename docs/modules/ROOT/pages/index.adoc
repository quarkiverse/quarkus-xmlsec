== Installation

If you want to use this extension, you need to add the `io.quarkiverse.xmlsec:quarkus-xmlsec` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.xmlsec</groupId>
    <artifactId>quarkus-xmlsec</artifactId>
    <version>{project-version}</version>
</dependency>
----
